# RETAILS_ANALYTICS
Retail Sales Analytics Platform | Databricks (Medallion Architecture)

• Solved messy retail data problem (inconsistent formats, duplicates, missing values)
• Built end-to-end pipeline using Bronze → Silver → Gold architecture
• Cleaned & standardized data (dates, prices, product names, null handling)
• Created single source of truth for sales, customers, and products
• Derived KPIs: Revenue, AOV, Repeat Rate, Return Rate, CLV, MoM Growth
• Identified top products, category performance, and customer behavior
• Built dashboard for business insights & decision-making
• Reduced manual reporting effort and improved data reliability
